# SIGA — Sistema de Gestión Fronteriza

Prototipo funcional del Sistema de Gestión Fronteriza (Paso Los Libertadores), desarrollado para la asignatura Ingeniería de Software (RQY1102).

## Descripción
SIGA es un sistema web de control migratorio fronterizo que permite registrar el ingreso y salida de pasajeros y vehículos, validar documentos, consultar antecedentes y generar reportes.

## Requerimientos implementados
- RF-01 al RF-20: Requerimientos funcionales
- RNF-01 al RNF-20: Requerimientos no funcionales
